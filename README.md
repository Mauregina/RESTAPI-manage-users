# RESTAPI 

Projetos visando aplicar conhecimento adquirido durante curso de REST API com Node.js e Express.

## APIs disponíveis

* Visualizar Pessoas
* Visualizar Pessoa por ID
* Cadastrar Pessoas
* Atualizar Pessoas
* Deletar Pessoas

## Ferramentas utilizadas na solução
```sh
npm install express nodemon mongoose dotenv
```

* Visual Studio SCode
* Mongo Atlas
